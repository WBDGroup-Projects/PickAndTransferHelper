Pick & Transfer Help

Overview

The Pick & Transfer Revit Plugin helps you transfer only selected standards between Revit files. This keeps your models clean and efficient.

Key Features

  • Transfer only specific parts of any standard
  • Supports loadable and system families
  • Enhances model quality and consistency

Installation Steps

  • Download the installer file
  • Close Revit before installing
  • Run the installer and follow: Next → Next → Install → Finish
  • Open Revit and go to WBD | Plugins

How to Use

  • Click on the plugin icon
  • Select a source Revit file from the dropdown
  • Click "Load Standards"
  • Select which standards to transfer
  • Click "Transfer"

Benefits

Keep your models lean by avoiding unnecessary data imports. Select only what you need and improve your workflow.
